# Что это
Это приложение, которое берет данные с newsapi.org и выводит их в виде списка новостей.

## Функциональность

- Получение списка новостей
- Просмотр новости
- Возможность перейти к полной новости на сайте источника
- Поддержка светлой и темной темы
